# 🚦 Traffic Sign Recognition System

## 📌 Project Overview

Traffic Sign Recognition System is a Deep Learning project that automatically identifies and classifies traffic signs from uploaded images. The system uses a Convolutional Neural Network (CNN) model trained on traffic sign datasets to recognize various road signs and provide their meaning.

The project helps drivers, learners, and intelligent transportation systems understand road signs accurately and efficiently through an interactive Streamlit web application.

## 🚀 Features

- Upload traffic sign images
- Automatic traffic sign detection
- Deep Learning-based classification
- Displays traffic sign name
- Shows prediction confidence score
- Provides sign meaning and explanation
- User-friendly Streamlit interface

## 📊 Supported Traffic Signs

The system can recognize signs such as:

- Speed Limit Signs
- Stop Sign
- No Entry
- Yield Sign
- Traffic Signals
- Road Work
- Pedestrian Crossing
- School Zone
- Turn Left / Right
- Priority Road
- And many more traffic signs

## ⚙️ How to Run

1. Clone the repository

```bash

```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
streamlit run app.py
```
